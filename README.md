# MVC pédagogique

Branche pour le développement d'un MVC simplifié pour l'apprentissage de la mise en place d'une architecture moderne avec PHP.

Fonctionnalités disponibles :

- CRUD d'articles avec modèle, vues et contrôleur pour la gestion d'un blog

Fonctionnalités à venir :

- CRUD de produits avec modèle, vues et contrôleur pour la gestion d'une boutique
- CRUD d'utilisateurs avec modèle, vues et contrôleur pour la gestion de comptes utilisateurs
